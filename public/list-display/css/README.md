# List Display Engine - CSS

This folder contains the CSS entrypoints for the list-based task manager app.

## Files

| File | Role |
|---|---|
| `task-manager.css` | Local app stylesheet entrypoint for `list-display/index.html` |

## Current wiring

`task-manager.css` is now the owned stylesheet for the list-display app.
The previous top-level shared stylesheet was moved here so the list UI keeps its HTML, CSS, and JS in one feature folder.
